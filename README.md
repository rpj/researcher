## researcher

Do some research

### irc bot

```
$ git clone https://github.com/rpj/researcher.git
$ cd researcher
$ python -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
$ python bot.py --nickname ... --server ... --channel ...
```

Prompt format: `research! query... ...`

### cli utility

```
$ python lib/research.py --help
```
